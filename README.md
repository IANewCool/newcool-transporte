# NewCool Transporte - Movilidad Chile

Portal educativo sobre transporte y movilidad en Chile.

## Demo

[https://transporte.newcool.io](https://transporte.newcool.io)

## Caracteristicas

- Buscador de plantas de revision tecnica
- Guia de licencias de conducir
- Calculadora de multas de transito
- Transporte publico
- Renovacion de documentos

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
